# SuperstoreDataAnalysis


## Project Overview

This project analyzes retail sales data from the Superstore dataset using Python, SQL, and Power BI.

The objective was to clean the raw dataset, perform business-focused sales analysis, and identify trends related to customer behavior, product performance, regional sales, and revenue generation.

## Tools Used

* Python (Pandas)
* PostgreSQL
* Power BI

---

## Data Cleaning

Data cleaning and preparation were performed using Python and Pandas.

### Cleaning Steps

* Standardized column names using snake_case formatting

* Converted date columns to proper datetime format

* Verified and handled missing values

* Checked for duplicate records

* Corrected data types where necessary

* Created new analytical columns:

  * shipping_days
  * order_year
  * order_month
  * order_quarter

* Exported a cleaned dataset for SQL analysis and visualization

---

## Business Questions Answered Using SQL

1. Which customers generated the highest revenue by sales?
2. How do sales compare across different categories?
3. What are the monthly sales trends?
4. Which regions contribute the most to total sales?
5. Which states generate the highest sales revenue?
6. How do sales differ across customer segments?
7. Which products are the top 10 best sellers by revenue?
8. What is the average order value?
9. Which shipping mode is used most often?
10. Which quarter generates the highest revenue?
11. Which cities generate the highest sales revenue?

---

## Key Insights

* Technology generated the highest sales revenue.
* The West region contributed the largest share of sales.
* Sales were concentrated among a relatively small group of customers.
* Average order value provided a useful benchmark for customer spending behavior.
* Customer segments showed different purchasing patterns and revenue contributions.
* Product sales were heavily concentrated among a limited number of top-performing products.
* Quarterly and monthly trends revealed seasonal variations in sales activity.
